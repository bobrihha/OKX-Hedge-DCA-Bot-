# OKX Hedge DCA Bot

Торговый бот для фьючерсов OKX с веб-интерфейсом управления. Поддерживает Hedge Mode, позволяя одновременно вести Long и Short позиции с использованием стратегии усреднения (DCA). Написан на Python (Flask).

## Особенности

- **Hedge Mode**: Одновременная торговля в Long и Short.
- **DCA (Dollar Cost Averaging)**: Стратегия усреднения позиций.
- **Веб-интерфейс**: Управление настройками и мониторинг через браузер.
- **Безопасность**: API ключи хранятся локально (требуется настройка).

## Установка и запуск

1.  Клонируйте репозиторий.
2.  Установите зависимости: `pip install -r okx_hedge_webapp/requirements.txt`
3.  Настройте `hedge_config.py` в папке бота или через веб-интерфейс.
4.  Запустите веб-приложение: `python okx_hedge_webapp/app.py`

## Структура

- `okx_hedge_webapp/`: Веб-интерфейс (Flask).
- `okx_hedge_bot1/`: Логика торгового бота.
